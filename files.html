flowp.files¶

from flowp.files import *
class flowp.files.cd(path)¶

Change working directory. Behave exacly like os.chdir, except that it can be used as context manager. Example:

with cd('some/path'):
    # do some operations
    pass

After exiting the context, return to the oryginal working directory. This also works but doesn’t return to the original working dir:

cd('some/path')
flowp.files.cp(src, dst, r=False)¶

Copy files or directories if r=True argument given

cp('dir/file.py', 'dir2/file.py')
cp('dir/*.py', 'dir2')
cp(['file1.py', 'file2.py'], 'dir')
cp('dir1', 'dir2', r=True)

class flowp.files.Watch(files, callback, sleep=0.2)¶

Create and start watch thread that will watch files and call given callable if some of the watch actions occurs.

Parameters:
  • files – files to watch given as glob path or list of glob paths
  • callback – callable(filename, action)
  • sleep – sleep value for watching loop
 
def callback(filename, action):
    if action == Watch.CHANGE:
        ...

w = Watch('*.py', callback)
w.wait()
CHANGE = 2¶

File changed action

DELETE = 3¶

File removed action

NEW = 1¶

New file action

stop(timeout=None)¶

Stop watch process. If timeout given stop process after given time.

stop_when(predicate, timeout=None)¶

Stop watch process when callable predicate() evaluates to True. If timeout given stop process after given time.

wait()¶

Hold process in the waiting state, keep watcher running
