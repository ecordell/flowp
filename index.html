Flowp 1.1¶

Flowp is a library which tries to bring the best ideas from Ruby / node.js world to Python making development more fun. For version 1.1 module flowp.testing is available which allows to write tests in a RSpec BDD style with minimum of magic and flowp.files module which brings convenient utils for files processing.

Installation¶

$ pip3 install flowp

Note

Only for Python >=3.3
